1. What We Collect

Tumeguard may process the following:

2. Why We Collect It

This data is used to:

  • Detect spam, toxic language, and malicious links
  • Enforce server moderation rules
  • Generate audit logs and bot actions

3. Third-Party Services

Tumeguard uses the following APIs solely for moderation purposes:

  • Google Perspective API
  • Google Safe Browsing
  • Google Web Risk

4. Data Retention

5. Your Rights

If you are a Discord server owner and wish to opt-out, restrict features, or request deletion of configuration data, contact the developer directly via Discord.
